# Contributing

Thanks for your interest in contributing!

## Ways to contribute
- Report bugs and propose features via Issues
- Submit small, focused PRs
- Improve docs and examples

## Getting started
1. Fork and clone the repo
2. Copy .env.example to .env.local and fill values
3. Install dependencies: `npm install`
4. Start dev server: `npm run dev`

## Pull Request checklist
- Keep PRs scoped and well-described
- Include screenshots or steps for UI changes
- Avoid committing secrets
- Ensure type checks pass and lint is clean

## Add yourself to Credits
Add an entry to `lib/contributors.ts` with:

```
{
	name: "Your Club or Name",
	affiliation: "Your University",
	url: "https://your-link",
	role: "What you contributed (optional)"
}
```
This list renders on the home screen.
